# Section 1: Getting Started

Welcome to the first section of the C++ course! This section introduces the core concepts every programmer needs before learning control flow, functions, and object-oriented programming.

## What You'll Learn

By the end of this section, you'll understand:

- How C++ programs are structured
- Primitive data types
- Variables and constants
- Naming conventions
- The `std` namespace
- Basic input and output
- Type casting
- Writing simple expressions
- Basic debugging techniques

## Topics

1. Introduction to C++
2. Program Structure
3. Data Types
4. Variables
5. Namespaces
6. Type Casting
7. Input and Output

## Prerequisites

None! This section is designed for complete beginners.

## Learning Goals

After completing this section, you should be able to:

- Create your first C++ program
- Store information in variables
- Choose the correct data type
- Convert between data types using type casting
- Read input from the user
- Display formatted output
- Understand the basic structure of a C++ program

## Competency Alignment

This section directly supports the first two competencies:

- Competency 1: Each lesson includes simple comments, and students are encouraged to write pseudocode before coding.
- Competency 2: The lessons cover integers, doubles, characters, booleans, strings, variables, operators, and type casting.

## Review Practice

- Write pseudocode for a small program that asks for a name and prints a greeting.
- Create a program that uses an integer, a double, a char, and a bool.
- Practice changing one data type into another using casting.
